ARCHERY.SERVICES — IMAGES
=========================

Drop royalty-free photos here (Unsplash, Pexels, Wikimedia Commons, etc.).
The site picks them up automatically by filename — no code changes needed.

Recognised filenames
--------------------
hero.jpg        Homepage hero background (wide archery action shot, ~1920x1080).
                NOTE: the admin panel's "Homepage Hero Image" upload overrides this.

You can also upload images directly in the Admin panel instead of using files:
  - Site Settings -> Homepage Hero Image  (sets the homepage hero)
  - Shop Products -> Add New Product -> Product Image  (per-product photo)

Licensing
---------
Only use images you have the right to use. Good free sources:
  - https://unsplash.com/s/photos/archery   (free, no attribution required)
  - https://www.pexels.com/search/archery/  (free, no attribution required)
Download, rename to hero.jpg, and place in this folder.
